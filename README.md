# 🌐 Portfólio Digital

Repositório centralizado para hospedagem e distribuição da minha página pessoal de apresentação profissional, competências técnicas e projetos em engenharia.

> 🚀 **Acesse o site oficial:** [https://JJ-LXXVI.github.io/portifolio/](https://JJ-LXXVI.github.io/portifolio/)

---

### 💻 Sobre o Projeto
Este site foi concebido estruturalmente como uma SPA (Single Page Application) estática, utilizando tecnologias front-end puras. O objetivo principal é servir como um hub de conexões e portfólio vivo para recrutadores, engenheiros e instituições de tecnologia.

A arquitetura de desenvolvimento seguiu as diretrizes de formação profissional aplicadas pelo educador Thiago Paulino Claro, integrando conceitos fundamentais de design responsivo e semântica web.

---

### 🛠️ Tecnologias Empregadas

* **Estruturação:** HTML5 Semântico (garantindo acessibilidade e SEO).
* **Estilização:** CSS3 nativo com técnicas de responsividade para dispositivos móveis.
* **Hospedagem & Infraestrutura:** GitHub Pages com deploy automatizado via Git.

---

### 📂 Arquivos do Repositório

* `Foto LinkedIn_resized.png`: Ativo de imagem utilizado para a identificação visual do perfil.
